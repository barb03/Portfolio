Liri App

Liri is an app that is like the iPhone Siri. It is a Language Interpretation and Recognition Interface. Liri takes in parameters and gives back data. The commands for Liri are: my-tweets returns a list of Twitter tweets, spotify-this-song returns song data, movie-this returns movie data, do-what-it-says returns command and data from a text file, and concert-this returns artist venue data.

Tech: Node.js, JavaScript, Spotify API, Twitter API, Bands in Town API and OMDB API. NPM Packages include spotify-web-api-node, dotenv, twitter, request, moment, chalk and axios.
